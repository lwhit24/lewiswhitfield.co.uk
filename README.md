# Lewis Whitfield Portfolio

A modern static portfolio site built for GitHub Pages. It uses plain HTML, CSS and a small amount of JavaScript, so there is no build step and no framework to install.

## Files

- `index.html` - the website content and structure
- `styles.css` - the full responsive visual design
- `script.js` - mobile navigation, active section highlighting and scroll reveal effects
- `assets/` - local images and certification badges

## Preview locally

Open `index.html` in your browser. The site is fully static, so it does not need a local server.

## Upload to GitHub

1. Create a new repository on GitHub.
2. Upload these files and folders to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/`
   - `README.md`
3. Commit the uploaded files to the `main` branch.

## Publish with GitHub Pages

1. Open the repository on GitHub.
2. Go to `Settings`.
3. Select `Pages` from the left-hand menu.
4. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
5. Set `Branch` to `main`.
6. Set the folder to `/root`.
7. Click `Save`.
8. Wait for GitHub to show the published Pages URL. It usually appears within a minute or two.

## Update the site later

Edit `index.html` for wording and links, replace images in `assets/`, then commit the changes to `main`. GitHub Pages will republish automatically.
